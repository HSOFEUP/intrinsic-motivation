 # Multi-modal Intrinsic Motivation for Robotic Exploration

 * Python 3
 * [PyTorch](http://pytorch.org/)

 ## Installation
```
git clone https://github.com/jmichaux/multimodal-curiosity.git
cd multimodal-curiosity
pip install -e .
```
